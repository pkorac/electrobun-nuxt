## Nuxt + Vue UI app

### Overview

A minimal Nuxt 4 starter app using Vue 3, Nuxt UI v4, and Tailwind CSS v4. The project uses **bun** as its package manager.

### Tech Stack

- **Framework:** Nuxt 4
- **UI Library:** Nuxt UI v4
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript (Vue `<script setup lang="ts">`)
- **Package Manager:** bun (see `bun.lock`)
- **Formatter:** Prettier (tabs for indentation — see `.prettierrc`)
